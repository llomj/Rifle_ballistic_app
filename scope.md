# Scopes — In-app catalog

All scopes currently available in the app. Expand this list to add more options.

## Structure (per entry)

| Field                    | Description                              |
|--------------------------|------------------------------------------|
| `id`                     | Unique identifier (slug)                 |
| `name`                   | Display name                             |
| `unit`                   | MIL or MOA                               |
| `clickValue`             | Angular increment per click (e.g. 0.1)   |
| `ffpOrSfp`               | FFP or SFP                               |
| `magnificationRange`      | e.g. 4-16x                               |
| `magnificationCalibration` | (optional) Magnification for SFP subtensions |
| `brand`                  | (optional) Brand name                    |
| `tubeDiameterMm`        | (optional) 25.4, 30, 34, 35, 36, 40     |
| `reticleType`            | (optional) Reticle model                 |
| `zeroStop`               | (optional) true if zero stop             |

## Reference: scope systems and options (databaseConstants)

- **Adjustment systems:** MOA, TMOA, SMOA, MIL, MRAD, IPHY
- **Click values (typical):** MOA: 0.25, 0.125, 0.5; MIL/MRAD: 0.1, 0.05; IPHY: 0.1, 0.2
- **Focal planes:** FFP, SFP
- **Tube diameters (mm):** 25.4 (1"), 30, 34, 35, 36, 40
- **Reticle types (examples):** Duplex, German #1/#4, Fine Crosshair, Post; Mil-Dot, Half Mil Dot, TMR, MSR, H59, Tremor3; Horus, EBR-2C, SCR, MSR2; BDC 600, BDC 800, Rapid-Z, Dead-Hold
- **Brands (tiers):** Premium (Swarovski, Zeiss, Schmidt & Bender, Kahles, Tangent Theta, Hensoldt); High-end (Nightforce, Leica, Steiner); Hunting (Vortex, Leupold, Burris, Meopta, Athlon, Trijicon); Budget (Bushnell, Hawke, Primary Arms, Sig Sauer)
